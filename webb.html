Fakta om Webbläsare

  • Webbläsare är, enligt Wikipedia "...ett program för att hämta, tolka och återge dokument kodade i HTML eller XHTML som lagts upp på webbservrar på Internet."

  • Den första webbläsaren skapades av Tim Berners-Lee år 1990.

  • Den första webbläsaren som lanserades var Mosaic vid NCSA år 1993.

  • Två år senare lanserades Netscape Navigator, som blev den första stora webbläsaren.
