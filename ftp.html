10 Fakta om FTP

  • FTP står för File Transfer Protocol.

  • Som namnet intyder, så är FTP ett filöverföringsprotokol.

  • FTP är ett kommandobaserat protokol för överföring av binära datafiler och text.

  • FTP är ett väldigt osäkert protokoll, till exempel är det mycket lättare att få tag på andras lösenord via FTP än det är med andra protokoll.

  • Trots detta var FTP rätt så länge standardprotokollet, förmodligen för att det var den ända.

  • Det finns inga inbyggda sökfuntioner, till skillnad från de flesta moderna protokoller.
